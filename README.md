# Structured Directory Maker

### Create directories from an input text file with a tree structure.

#### Example
sample_tree.txt
```
Movies
  John Wick
  Pulp Fiction
Photos
  Canon
    Beach
    Park
Work
  Presentations
```

#### Usage:
```
py main.py C:\Users\john\tree.txt (input) C:\Users\john\Documents (output)
```
